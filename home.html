Front-end web development, also known as client-side development is the practice of producing HTML, CSS and JavaScript for a website or Web Application so that a user can see and interact with them directly. The challenge associated with front end development is that the tools and techniques used to create the front end of a website change constantly and so the developer needs to constantly be aware of how the field is developing.

The objective of designing a site is to ensure that when the users open up the site they see information in a format that is easy to read and relevant. This is further complicated by fact that users now use large variety of devices with varying screen sizes resolutions thus forcing designer take into consideration these aspects when designing site. They need ensure systems cross-platform correctly different browsers (cross-browser), different operating their (site comes up) different devices (cross-device), which requires careful planning on side developer.
